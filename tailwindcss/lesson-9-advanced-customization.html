Lesson 9: Advanced Customization

Extend the Theme

Use theme.extend to add colors, fonts, spacing, and more without replacing defaults.

module.exports = {
  theme: {
    extend: {
      colors: { brand: { 500: "#3c66ff", 600: "#284be0" } },
      fontFamily: { display: ['"Inter var"', "Inter", "ui-sans-serif"] }
    }
  }
}

@layer & @apply

Build reusable components and utilities in your CSS:

@layer components {
  .btn { @apply inline-flex items-center gap-2 px-4 py-2 rounded-lg 
         bg-brand-500 text-white hover:bg-brand-600; }
}

Arbitrary Values & Variants

Reach for one-off precision when needed:

<div class="w-[37rem] rounded-[22px] bg-brand-500/10"></div>

Plugins

Use community plugins or author your own with the plugin API.

npm i @tailwindcss/typography @tailwindcss/forms @tailwindcss/aspect-ratio

Build-Time Controls

  • Safelist: keep classes from purging via safelist.
  • Prefix: avoid collisions using prefix: "tw-".
  • Important: set important: true or scope via a selector.
  • Core Plugins: disable features you don't need.

Tip: For production, configure content globs precisely and prefer tokens over arbitrary values for consistency.
