## ğŸ§¾ Step 3: Create project folders and CSS entry file

If you're using **PowerShell**, run:

```powershell
mkdir src, public
'@import "tailwindcss";' | Out-File -FilePath src\input.css -Encoding utf8
```

If you're using **Command Prompt (CMD)**, run:

```cmd
mkdir src public
echo @import "tailwindcss"; > src\input.css
```

Or create manually:

**src/input.css**
```css
@import "tailwindcss";
```

## âš™ï¸ Step 4: Add build scripts in `package.json`

Open `package.json` and add these scripts inside the `"scripts"` section:

```json

  "scripts": {
    "dev": "npx @tailwindcss/cli -i ./src/input.css -o ./public/tailwind.css --watch",
    "build": "npx @tailwindcss/cli -i ./src/input.css -o ./public/tailwind.css"
  }
}
```

- `npm run dev` â†’ watch for changes and rebuild automatically
