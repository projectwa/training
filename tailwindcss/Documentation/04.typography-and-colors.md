# 4. Typography and Colors — Tailwind CSS

This chapter explains how to control **typography** (font, size, line-height, spacing, alignment, decoration) and **colors** (text, background, borders, gradients, opacity, dark mode) using Tailwind CSS. It includes a quick-reference table and runnable examples.

> If you’re using Tailwind via a build step, all classes shown below work as long as your content files are scanned by Tailwind’s `content` setting. For the downloadable HTML example in this lesson, we use the Tailwind CDN for simplicity.

---

## 1) Typography

### 1.1 Font family
- `font-sans`, `font-serif`, `font-mono`

```html
<p class="font-sans">Sans</p>
<p class="font-serif">Serif</p>
<p class="font-mono">Mono</p>
```

### 1.2 Font size & line-height
- Sizes: `text-xs` → `text-9xl`
- Combine with leading: `leading-none`, `leading-tight`, `leading-snug`, `leading-normal`, `leading-relaxed`, `leading-loose`
- Arbitrary values: `text-[22px]`, `leading-[1.35]`

```html
<p class="text-base leading-relaxed">Base text with relaxed line height.</p>
<p class="text-2xl md:text-4xl leading-tight">Responsive size with tight leading.</p>
```

### 1.3 Font weight & style
- Weights: `font-thin` → `font-black`
- Italic: `italic` / `not-italic`

```html
<p class="font-semibold">Semi-bold</p>
<p class="italic">Italic text</p>
```

### 1.4 Letter-spacing & word-break
- Tracking (letter-spacing): `tracking-tighter`, `tracking-tight`, `tracking-normal`, `tracking-wide`, `tracking-wider`, `tracking-widest`, or `tracking-[.02em]`
- Text wrap & break: `break-normal`, `break-words`, `break-all`, `truncate`, `line-clamp-*` (add plugin for clamp)

```html
<p class="tracking-wide">Spaced letters</p>
<p class="truncate w-40">This is a super long line that will be truncated…</p>
```

### 1.5 Text alignment & transform
- Alignment: `text-left`, `text-center`, `text-right`, `text-justify`
- Transform: `uppercase`, `lowercase`, `capitalize`, `normal-case`

```html
<p class="text-center uppercase">Centered and uppercase</p>
```

### 1.6 Text decoration & emphasis
- Decoration: `underline`, `overline`, `line-through`, `no-underline`
- Decoration style: `decoration-dotted`, `decoration-wavy`, etc. (with modern Tailwind)
- Decoration thickness & offset: `decoration-2`, `underline-offset-4`
- Emphasis (selection): `selection:bg-yellow-200 selection:text-black`

```html
<p class="underline decoration-wavy decoration-2 underline-offset-4">
  Fancy underline
</p>
```

### 1.7 Lists & quotes
- Lists: `list-disc`, `list-decimal`, `list-none`, and `list-inside` / `list-outside`
- Quotes: `italic border-l-4 pl-4` (utility combo)

```html
<ul class="list-disc list-inside space-y-1">
  <li>First</li>
  <li>Second</li>
</ul>

<blockquote class="italic border-l-4 pl-4 text-slate-700">
  Utility-first CSS is powerful.
</blockquote>
```

### 1.8 The `prose` class (Typography plugin)
If you enable `@tailwindcss/typography`, you get opinionated defaults for rich text:
- Add the plugin in `tailwind.config.js`, then use `<article class="prose">...</article>`.
- Great for blog posts and CMS content.

## 2) Colors

Tailwind ships with an extensive color palette (e.g., `slate`, `gray`, `zinc`, `neutral`, `stone`, `red`, `orange`, `amber`, `yellow`, `lime`, `green`, `emerald`, `teal`, `cyan`, `sky`, `blue`, `indigo`, `violet`, `purple`, `fuchsia`, `pink`, `rose`).

### 2.1 Text & background colors
- Text: `text-blue-600`, `text-slate-700`, `text-white`
- Background: `bg-blue-50`, `bg-white/80` (with opacity), `bg-[var(--brand)]` (arbitrary)

```html
<p class="text-slate-700">Readable slate text</p>
<div class="bg-blue-50 p-3 rounded">Soft blue background</div>
```

### 2.2 Borders, divide, ring & outline
- Border color: `border-red-500`
- Divide color between children: `divide-gray-200`
- Ring color (focus): `ring-2 ring-blue-500`
- Outline: `outline outline-2 outline-offset-2 outline-emerald-500`

```html
<button class="border border-gray-300 ring-2 ring-transparent focus:ring-blue-500 px-3 py-1 rounded">
  Focus me
</button>
```

### 2.3 Gradients
- Use `bg-gradient-to-r`/`l`/`t`/`b` etc. with `from-*`, `via-*`, `to-*`

```html
<div class="h-16 bg-gradient-to-r from-indigo-500 via-purple-500 to-pink-500 rounded"></div>
```

### 2.4 Opacity utilities
- `opacity-0` → `opacity-100`
- Per color using slash: `bg-blue-500/20`, `text-black/60`

```html
<p class="text-black/60">Dimmed text</p>
```

### 2.5 Dark mode
- Configure `darkMode: 'class'` in `tailwind.config.js` (recommended).
- Use `dark:` variant: `dark:text-slate-200`, `dark:bg-slate-900`.
- Toggling the `dark` class on `<html>` or a wrapper switches themes.

```html
<div class="bg-white text-slate-900 dark:bg-slate-900 dark:text-slate-200 p-4 rounded">
  Auto-styles in dark mode.
</div>
```

### 2.6 Arbitrary values & CSS vars
- Any color: `text-[#0ea5e9]`, `bg-[rgb(10,10,10)]`
- With CSS vars: `text-[var(--brand-text)]`

```html
<div style="--brand:#14b8a6">
  <p class="text-[var(--brand)]">Brand text via CSS var</p>
</div>
```

## 3) Quick Reference

| Category | Common Classes | Notes |
|---|---|---|
| Font | `font-sans`, `font-serif`, `font-mono` | Set family |
| Size | `text-xs` → `text-9xl`, `text-[22px]` | Use responsive variants like `md:text-2xl` |
| Weight | `font-light` → `font-black` | |
| Leading | `leading-none` → `leading-loose`, `leading-[1.4]` | |
| Tracking | `tracking-tight` → `tracking-widest`, `tracking-[0.02em]` | |
| Align | `text-left/center/right/justify` | |
| Transform | `uppercase`, `capitalize`, `normal-case` | |
| Decoration | `underline`, `line-through`, `decoration-*`, `underline-offset-*` | |
| Lists | `list-disc/decimal/none`, `list-inside/outside` | |
| Text Color | `text-slate-700`, `text-blue-600`, `text-white/80` | |
| Background | `bg-slate-50`, `bg-blue-500/20`, gradients | `bg-gradient-to-r from-* via-* to-*` |
| Borders/Rings | `border-*-*`, `ring-*`, `outline-*` | Focus states with `focus:` |
| Dark Mode | `dark:text-*`, `dark:bg-*` | Set `darkMode: 'class'` |
