# 3. Layout and Positioning in Tailwind CSS

Tailwind CSS provides powerful **utility classes** for controlling layout and positioning of elements.  
It allows you to manage **display**, **flexbox**, **grid**, **positioning**, **spacing**, and **alignment** without writing custom CSS.

---

## ðŸš€ Key Concepts

### 1. Display
Controls how an element is displayed on the page.

| Class | Description |
|--------|--------------|
| `block` | Makes the element a block-level element |
| `inline-block` | Inline-level block element |
| `flex` | Enables Flexbox layout |
| `grid` | Enables Grid layout |
| `hidden` | Hides the element |

---

### 2. Positioning
Defines how an element is positioned in the document.

| Class | Description |
|--------|--------------|
| `static` | Default positioning |
| `relative` | Allows positioning relative to itself |
| `absolute` | Positions element relative to nearest positioned ancestor |
| `fixed` | Positions element relative to the viewport |
| `sticky` | Sticks element when scrolling |

Example:
```html
<div class="relative bg-gray-200 p-6">
  <div class="absolute top-2 right-2 bg-blue-500 text-white px-3 py-1 rounded">
    Absolute Box
  </div>
  <p>This is a relative container. The blue box is positioned absolutely inside it.</p>
</div>
```

---

### 3. Flexbox Layout
Tailwind makes it easy to use Flexbox for aligning and distributing items.

| Class | Purpose |
|--------|----------|
| `flex` | Enables flex container |
| `flex-row` / `flex-col` | Controls direction |
| `justify-*` | Horizontal alignment (`start`, `center`, `between`, `end`) |
| `items-*` | Vertical alignment (`start`, `center`, `end`, `stretch`) |
| `gap-*` | Space between items |

Example:
```html
<div class="flex justify-between items-center bg-gray-100 p-4 rounded-lg shadow">
  <div class="bg-blue-500 text-white px-4 py-2 rounded">Left</div>
  <div class="bg-green-500 text-white px-4 py-2 rounded">Center</div>
  <div class="bg-red-500 text-white px-4 py-2 rounded">Right</div>
</div>
```

---

### 4. Grid Layout
Tailwind provides an easy way to build responsive grid layouts.

| Class | Description |
|--------|--------------|
| `grid` | Enables grid layout |
| `grid-cols-*` | Defines number of columns |
| `gap-*` | Controls gap between grid items |
| `col-span-*` | Controls how many columns an item spans |

Example:
```html
<div class="grid grid-cols-1 sm:grid-cols-2 lg:grid-cols-3 gap-4">
  <div class="bg-blue-200 p-4 rounded">Item 1</div>
  <div class="bg-green-200 p-4 rounded">Item 2</div>
  <div class="bg-yellow-200 p-4 rounded">Item 3</div>
</div>
```

---

### 5. Spacing (Margin & Padding)
| Class | Description |
|--------|--------------|
| `m-*` | Margin |
| `p-*` | Padding |
| `mx-*` / `my-*` | Horizontal / Vertical margin |
| `px-*` / `py-*` | Horizontal / Vertical padding |

Example:
```html
<div class="m-4 p-4 bg-purple-200 rounded-lg">
  <p>Margin and padding applied using Tailwind utilities.</p>
</div>
```

Below is a complete HTML example demonstrating **layout and positioning** features together.

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Tailwind Layout & Positioning</title>

</head>
<body class="bg-gray-50 p-6">

  <!-- Navbar -->
  <nav class="bg-gray-800 p-4 mb-6">
    <div class="container mx-auto flex justify-between items-center text-white">
      <h1 class="text-xl font-bold">My Website</h1>
      <div class="flex gap-4">
        <a href="#" class="hover:text-blue-300">Home</a>
        <a href="#" class="hover:text-blue-300">About</a>
        <a href="#" class="hover:text-blue-300">Contact</a>
      </div>
    </div>
  </nav>

  <!-- Hero Section -->
  <section class="relative bg-blue-100 p-6 rounded-lg shadow-inner mb-6">
    <h2 class="text-2xl font-semibold mb-2">Welcome to Tailwind Layout!</h2>
    <p>Learn how to build modern layouts using simple utility classes.</p>
    <button class="absolute top-4 right-4 bg-blue-600 text-white px-3 py-1 rounded-lg">Learn More</button>
  </section>

  <!-- Grid Section -->
  <section class="grid grid-cols-1 sm:grid-cols-2 lg:grid-cols-3 gap-4">
    <div class="bg-yellow-200 p-4 rounded-lg shadow">Grid Item 1</div>
    <div class="bg-green-200 p-4 rounded-lg shadow">Grid Item 2</div>
    <div class="bg-red-200 p-4 rounded-lg shadow">Grid Item 3</div>
  </section>

</body>
</html>
```
