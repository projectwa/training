# 5. Borders, Effects, and Backgrounds (Tailwind CSS)

This chapter covers three big styling buckets in Tailwind:
- **Borders**: width, color, radius, style, divide, outline vs. ring.
- **Effects**: shadows, opacity, filters (blur, brightness, etc.), backdrop filters, mix-blend, will-change.
- **Backgrounds**: solid colors, gradients, images, size/position/repeat/attachment.

All examples work with the Tailwind CDN or a compiled build. Copy–paste the snippets into an `.html` file with Tailwind enabled.

---

## Prerequisites (quick)
If you're not using a build step, add Tailwind via CDN in your HTML `<head>`:
```html
<script src="https://cdn.tailwindcss.com"></script>
```
If you are using a build step, make sure your generated CSS is linked:
```html
<link href="/public/tailwind.css" rel="stylesheet">
```

## A. Borders

### 1) Border Width & Sides
- `border` (alias for `border-1` in Tailwind CLI) → typically use explicit sizes like `border-2`, `border-4`, `border-8`
- Per-side: `border-t`, `border-r`, `border-b`, `border-l`
- Remove: `border-0`, `border-t-0`, etc.

```html
<div class="p-4 border-2">All sides</div>
<div class="p-4 border-t-4">Top only</div>
<div class="p-4 border-x-4">Left & Right</div>
```

### 2) Border Color
Use any palette color: `border-gray-300`, `border-blue-500`, `border-red-600/50` (with opacity).
```html
<div class="p-4 border-2 border-emerald-500">Emerald border</div>
<div class="p-4 border-2 border-red-500/50">50% opacity</div>
```

### 3) Border Radius (Rounded Corners)
- `rounded-none`, `rounded-sm`, `rounded`, `rounded-md`, `rounded-lg`, `rounded-xl`, `rounded-2xl`, `rounded-3xl`, `rounded-full`
- Per-corner/axis: `rounded-t-lg`, `rounded-b-xl`, `rounded-l-2xl`

```html
<div class="p-4 border rounded-lg">Rounded box</div>
<div class="p-4 border rounded-full">Pill / circle (with fixed height)</div>
```

### 4) Border Style
- `border-solid`, `border-dashed`, `border-dotted`, `border-double`, `border-none`
```html
<div class="p-4 border-2 border-dashed border-indigo-500">Dashed</div>
```

### 5) Outline vs. Ring (Focus styles)
- **Outline**: `outline`, `outline-none`, `outline-2`, `outline-blue-500`
- **Ring** (box-shadow-based): `ring`, `ring-2`, `ring-offset-2`, `ring-blue-500`  
  Rings layer *outside* borders and can have an offset.
```html
<button class="px-4 py-2 rounded-lg border focus:outline-none focus:ring-2 focus:ring-blue-500 focus:ring-offset-2">
  Focus me
</button>
```

### 6) Divide Utilities (Between children)
Add borders between flex/grid children: `divide-x`, `divide-y`, and colors/opacity: `divide-gray-300`, `divide-dashed`.
```html
<div class="flex divide-x divide-gray-300">
  <div class="p-2">A</div>
  <div class="p-2">B</div>
  <div class="p-2">C</div>
</div>
```

## B. Effects

### 1) Shadows
- `shadow-sm`, `shadow`, `shadow-md`, `shadow-lg`, `shadow-xl`, `shadow-2xl`, `shadow-inner`, `shadow-none`
- Colored shadows via utilities like `shadow-black/20` (Tailwind v3.4+ requires plugin or use custom class in config; otherwise use rings or backdrop tricks).

```html
<div class="p-6 rounded-lg shadow-lg">Card with shadow</div>
<div class="p-6 rounded-lg shadow-inner">Inset shadow</div>
```

### 2) Opacity
- For any color utility add `/opacity` like `bg-blue-500/20`
- For element opacity: `opacity-0`, `opacity-50`, `opacity-100`
```html
<img class="opacity-75 hover:opacity-100 transition" src="..." alt="">
```

### 3) CSS Filters
- `blur`, `brightness`, `contrast`, `grayscale`, `hue-rotate`, `invert`, `saturate`, `sepia`
- Require `filter` base utility in older Tailwind versions; in v3+, shorthand utilities enable filter automatically.
```html
<img class="grayscale hover:grayscale-0 transition" src="..." alt="">
<div class="backdrop-blur-md bg-white/60 p-4 rounded-xl">Frosted glass panel</div>
```

### 4) Mix Blend & Background Blend
- `mix-blend-multiply`, `mix-blend-screen`, `mix-blend-overlay`, `mix-blend-normal`
- `bg-blend-multiply`, etc., when combining multiple backgrounds
```html
<div class="mix-blend-multiply bg-yellow-300 p-6">
  On top of an image → color blends with content behind
</div>
```

### 5) Will-Change (Performance Hint)
- `will-change-auto`, `will-change-transform`, `will-change-scroll`
```html
<div class="will-change-transform hover:scale-105 transition">Optimized hover scale</div>
```

## C. Backgrounds

### 1) Solid Colors & Opacity
```html
<div class="bg-slate-100 p-4">Subtle background</div>
<div class="bg-blue-500/10 p-4">Tint with 10% opacity</div>
```

### 2) Gradients
- Use `bg-gradient-to-{r|l|t|b|tr|tl|br|bl}` + `from-*`, `via-*`, `to-*`
```html
<div class="bg-gradient-to-r from-indigo-500 via-sky-500 to-emerald-500 text-white p-6 rounded-xl">
  Nice gradient
</div>
```

### 3) Background Images
- `bg-[url('/path.jpg')]`
- Size: `bg-cover`, `bg-contain`, `bg-auto`
- Position: `bg-center`, `bg-left`, `bg-right`, `bg-top`, `bg-bottom`
- Repeat: `bg-no-repeat`, `bg-repeat`, `bg-repeat-x`, `bg-repeat-y`
- Attachment: `bg-fixed`, `bg-local`, `bg-scroll`
```html
<section class="h-48 bg-[url('https://images.unsplash.com/photo-1500530855697-b586d89ba3ee?q=80&w=1200&auto=format&fit=crop')] bg-cover bg-center rounded-xl"></section>
```

### 4) Multiple Background Tricks
You can layer gradient + image (image on top by default). To tint an image, combine a gradient with `bg-blend-multiply`:
```html
<section class="h-56 rounded-xl bg-gradient-to-r from-indigo-700 to-sky-500 bg-[url('https://images.unsplash.com/photo-1500530855697-b586d89ba3ee?q=80&w=1200&auto=format&fit=crop')] bg-cover bg-center bg-blend-multiply"></section>
```

## D. Responsive & State Variants

All utilities work with responsive prefixes (`sm:`, `md:`, `lg:`, `xl:`, `2xl:`) and state variants like `hover:`, `focus:`, `active:`, `disabled:`, `group-hover:`, `aria-*`, `data-*`.
```html
<button class="bg-gray-100 hover:bg-gray-200 md:bg-gray-50 md:hover:bg-gray-100 border md:border-0 rounded-lg px-4 py-2">
  Responsive + hover variants
</button>
```

## F. Reference Cheat Sheet

- **Borders**
  - Width: `border`, `border-0/2/4/8`, sides: `border-x`, `border-y`, `border-t-4`, …
  - Color: `border-{color}-{shade}`, add `/opacity`
  - Radius: `rounded`, `rounded-{sm|md|lg|xl|2xl|3xl|full}`, per-side/corner variants
  - Style: `border-solid|dashed|dotted|double|none`
  - Divide: `divide-x|y` + color/style, `divide-dashed`
  - Outline vs. Ring: `outline-*` vs `ring-*` + `ring-offset-*`

- **Effects**
  - Shadow: `shadow-sm|md|lg|xl|2xl|inner|none`
  - Opacity: `opacity-0–100`, color opacity via `/`
  - Filters: `blur`, `brightness`, `contrast`, `grayscale`, `hue-rotate`, `invert`, `saturate`, `sepia`
  - Backdrop: `backdrop-blur`, `backdrop-brightness`, etc.
  - Blend: `mix-blend-*`, `bg-blend-*`
  - Will change: `will-change-*`

- **Backgrounds**
  - Color: `bg-{color}-{shade}` with `/opacity`
  - Gradient: `bg-gradient-to-*` + `from/via/to`
  - Image: `bg-[url('...')]` + `bg-cover|contain|center|no-repeat|fixed`

> Tip: Combine **transition** (`transition`, `duration-200`, `ease-in-out`) to make hover/focus states feel smooth.
