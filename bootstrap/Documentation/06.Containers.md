# Bootstrap Containers

Containers are one of the **most basic and essential** layout elements in Bootstrap.  
They are used to contain, pad, and align your page content within a given viewport width.

---

## ğŸ§± Types of Containers

Bootstrap provides **three main types** of containers:

| Type | Class | Description |
|------|--------|-------------|
| Fixed-width | `.container` | Has a **responsive fixed width** that changes at each breakpoint. |
| Fluid | `.container-fluid` | Always spans **100% width** across all breakpoints. |
| Responsive | `.container-{breakpoint}` | Becomes fixed-width at the specified breakpoint and fluid before that. Example: `.container-md` |
