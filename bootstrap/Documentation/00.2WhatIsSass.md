**Sass** (short for **Syntactically Awesome Style Sheets**) is a **CSS preprocessor** â€” a tool that extends the capabilities of standard CSS. It allows developers to write cleaner, more efficient, and more maintainable stylesheets.

## ðŸ§  Key Features of Sass

### 1. Variables
Define reusable values like colors, fonts, or spacing.
```scss
$primary-color: #007bff;

body {
  background-color: $primary-color;
}
```

### 2. Nesting
Write CSS in a hierarchical way that mirrors HTML structure.
```scss
nav {
  ul {
    list-style: none;
  }
  a {
    text-decoration: none;
  }
}
```

### 3. Partials & Imports
Split your CSS into smaller, organized files and import them into one main file.
```scss
// _buttons.scss
.btn {
  padding: 10px;
}

// main.scss
@import 'buttons';
```

### 4. Mixins
Reuse blocks of CSS with parameters â€” like functions for CSS.
```scss
@mixin flex-center {
  display: flex;
  justify-content: center;
  align-items: center;
}

.container {
  @include flex-center;
}
```

### 5. Inheritance (Extends)
Share common styles between selectors.
```scss
.btn {
  padding: 10px;
  border-radius: 5px;
}

.btn-primary {
  @extend .btn;
  background-color: blue;
}
```

### 6. Mathematical & Logical Operations
You can do calculations directly in Sass.
```scss
.box {
  width: 100% / 3;
}
```

## ðŸ†š Sass vs SCSS

| Format | File Extension | Syntax Style | Example |
|---------|----------------|---------------|----------|
| **Sass** | `.sass` | Indentation-based (no curly braces or semicolons) | `nav ul\n  margin: 0` |
| **SCSS** | `.scss` | Similar to regular CSS (uses `{}` and `;`) | `nav ul { margin: 0; }` |

> **Bootstrap** uses the **SCSS** syntax.

## ðŸŽ¨ Example: Customizing Bootstrap with Sass

You can override Bootstrapâ€™s default Sass variables before importing its core styles. For example, to change the primary color:

```scss

// 1. Override Bootstrap variable
$primary: #ff5733;

// 2. Import Bootstrap's main SCSS

// 3. Add your custom styles

```

Then compile it:
```bash

```

And link it in your HTML:
```html
<link rel="stylesheet" href="custom.css">
```
