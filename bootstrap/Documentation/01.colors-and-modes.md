## ğŸŒ— 3. Color Modes (Light & Dark)

Bootstrap 5.3 introduced a **color mode system**, allowing easy switching between **light** and **dark** themes using CSS variables.

### Default Light Mode
By default, Bootstrap uses **light mode** colors.

```html
<body data-bs-theme="light">
  <div class="bg-light text-dark p-3">Light Mode Example</div>
</body>
```

### Dark Mode
To enable dark mode, change the `data-bs-theme` attribute:

```html
<body data-bs-theme="dark">
  <div class="bg-dark text-light p-3">Dark Mode Example</div>
</body>
```

Bootstrap automatically adapts components and utilities to match the selected theme.

## ğŸŒ“ 4. Auto Theme Switching with JavaScript

You can toggle between modes dynamically using JavaScript:

```html
<button id="toggleMode" class="btn btn-secondary">Toggle Theme</button>

<script>
const body = document.body;
document.getElementById('toggleMode').addEventListener('click', () => {
  const currentTheme = body.getAttribute('data-bs-theme');
  body.setAttribute('data-bs-theme', currentTheme === 'dark' ? 'light' : 'dark');
});
</script>
```
