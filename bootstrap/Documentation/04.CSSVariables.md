# CSS Variables

## Introduction

CSS Variables (also known as *custom properties*) are user-defined values that store reusable information directly in CSS.  
They help make stylesheets more **maintainable**, **scalable**, and **easy to update** â€” especially in large projects.

CSS Variables are declared using the `--` prefix and accessed using the `var()` function.

---

## Syntax

```css
:root {
  --main-color: #3498db;
  --font-size: 16px;
}
```

You can then use these variables anywhere in your CSS:

```css
body {
  background-color: var(--main-color);
  font-size: var(--font-size);
}
```

## Dynamic Theming Example (JavaScript)

CSS Variables can be changed dynamically with JavaScript:

```js
// Change a single variable
document.documentElement.style.setProperty('--primary-color', '#e74c3c');

// Toggle between predefined themes by flipping a data attribute
const root = document.documentElement;
root.setAttribute('data-theme', root.getAttribute('data-theme') === 'dark' ? 'light' : 'dark');
```

This allows for **live theme switching** without reloading the page.

## Difference Between CSS Variables and SCSS Variables

| Feature | **CSS Variables** | **SCSS Variables** |
|----------|-------------------|--------------------|
| **Syntax** | `--variable-name: value;` and used with `var(--variable-name)` | `$variable-name: value;` |
| **Processing** | Runs in the **browser at runtime** | Processed by the **Sass compiler at build time** |
| **Dynamic Updates** | Can be changed dynamically using JavaScript | Cannot be changed once compiled |
| **Scope** | Follows normal CSS cascade and inheritance | Follows Sass scoping (block-level or global) |
| **Browser Support** | Supported in modern browsers (CSS3+) | Requires Sass preprocessor, outputs regular CSS |
| **Usage** | Good for **themes, color modes, or runtime changes** | Great for **complex logic, functions, and mixins** |
| **Example** | `:root { --primary: blue; } color: var(--primary);` | `$primary: blue; color: $primary;` |

## Example Comparison

### CSS Variable

```css
:root {
  --primary-color: #007bff;
}

button {
  background: var(--primary-color);
}
```

### SCSS Variable

```scss
$primary-color: #007bff;

button {
  background: $primary-color;
}
```

## Summary

| Concept | Description |
|----------|--------------|
| Declaration | `--variable-name: value;` |
| Usage | `var(--variable-name)` |
| Scope | Inherits from parent or `:root` |
| Change dynamically | `element.style.setProperty('--name', value)` |
| SCSS alternative | `$variable-name` (compiled at build time) |
