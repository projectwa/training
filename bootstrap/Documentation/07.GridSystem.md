# Bootstrap Grid System (v5)

The Bootstrap grid is a **flexbox-based 12â€‘column** layout system. It uses a hierarchy of:
**containers** â†’ **rows** â†’ **columns** to build responsive layouts.

## Core Concepts

- **12 columns:** A row can hold up to 12 column units. Columns that exceed 12 wrap to the next line.
- **Responsive breakpoints:** Control layout per device size using classes like `-sm-`, `-md-`, `-lg-`, `-xl-`, `-xxl-`.
- **Flexbox:** Columns are flex items inside `.row` (which is a flex container).
- **Gutters:** Spacing between columns; controlled by `.g-*`, `.gx-*` (horizontal), `.gy-*` (vertical). Default gutter width is `1.5rem` horizontally.
- **Auto layout:** Use `.col` for equalâ€‘width columns that auto size.
- **Nesting:** Place a `.row` inside a `.col-*` and continue the pattern.

### Breakpoints (defaults)

| Infix | Min width | Typical use |
|---|---:|---|
| *(none)* | 0px | extraâ€‘small phones |

### Container types (quick recap)

- `.container` â€“ responsive fixedâ€‘width at each breakpoint
- `.container-{breakpoint}` â€“ fixed until that breakpoint, then fluid
- `.container-fluid` â€“ full width, always

> Grid columns **must** be placed inside a `.row`, and rows should live inside a container.

## Common Column Classes

- Equal width: `.col` (auto)
- Fixed fraction: `.col-1` â€¦ `.col-12`
- Responsive: `.col-sm-6`, `.col-md-4`, etc.
- Auto width by content: `.col-auto`
- Offsets: `.offset-*-*` (e.g., `.offset-md-2`)
- Order: `.order-0` â€¦ `.order-5`, `.order-first`, `.order-last`
- Row columns: `.row-cols-2`, `.row-cols-md-3` to set column count without specifying widths

## Alignment & Distribution (Flex utilities)

Apply on `.row` (or `.d-flex` parent):

- Horizontal: `.justify-content-start|center|end|between|around|evenly`
- Vertical: `.align-items-start|center|end|stretch`
- Perâ€‘column override: `.align-self-start|center|end|stretch` on a column

## Gutters

- Both axes: `.g-0` â€¦ `.g-5`
- Horizontal only: `.gx-0` â€¦ `.gx-5`
- Vertical only: `.gy-0` â€¦ `.gy-5`

> Use `.g-0` to remove gutters entirely.

## Tips & Gotchas

- Put only `.col-*` elements as **direct** children of `.row`.
- Avoid setting padding on `.row`; adjust gutters via `.g*` classes instead.
- When using backgrounds/borders to visualize the grid, add inner wrappers so gutters donâ€™t doubleâ€‘up.
- Use `.img-fluid` for responsive images inside columns.
- Combine with display utilities like `.d-none d-md-block` for show/hide per breakpoint.

- Offset: `.offset-md-2`
- Order: `.order-1`, `.order-md-3`
- Alignment: `.align-items-center` on row, `.align-self-end` on a column
